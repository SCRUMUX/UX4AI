# Legacy CSS файлы тем

Эта директория содержит старые CSS-файлы тем, которые были актуальны на момент создания, но теперь устарели.

**Статус:** Архив / Legacy  
**Дата переноса:** Фаза 3 — Система тем и цветовая архитектура

## Содержимое

- `theme-variables.css` — старые CSS-переменные для тёмной и светлой тем. Дублирует функциональность `styles/tokens.css`, но с другими именами переменных (`--bg-primary`, `--text-primary` и т.д.).
- `simple-themes.css` — простые темы (светлая, неоновая). Содержит дублирующие определения цветов.

## Почему они здесь?

Эти файлы не используются в текущей версии проекта:
- ❌ Не подключены в `index.html`
- ❌ Не используются в коде
- ❌ Все необходимые цвета определены в `core/theme-colors.js` и `styles/tokens.css`

## Актуальная система тем

**Источники истины:**
- **`core/theme-colors.js`** — единый источник всех цветов (UI + 3D сцена)
- **`themes/ui-theme-config.js`** — конфигурации Dark/Light тем
- **`ui/theme-controller.js`** — единая точка применения тем
- **`styles/tokens.css`** — CSS-переменные для UI (маппятся на токены из `core/theme-colors.js`)

**Документация:**
- **[THEME-SYSTEM.md](../THEME-SYSTEM.md)** — архитектура системы тем
- **[docs/COLOR_MAP.md](../docs/COLOR_MAP.md)** — документация всех цветовых токенов
- **[THEME-CUSTOMIZATION-GUIDE.md](../THEME-CUSTOMIZATION-GUIDE.md)** — руководство по настройке тем
